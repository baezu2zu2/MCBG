##1.기능

----------


>####사람이 죽었을 떄
>- 아이템을 상자에 넣은 채 사망
>- 사람한테 킬 시 관전모드로 변경


>####월드보더 
>- 월드보더를 넘어가면 자동으로 맨 위로 tp됨
>- 월드보더 중심은 원래 월드보더 안을 벗어나지 않는 범위에서 랜덤으로 이동함
>- 월드보더의 간격
>  - 1페이즈 10분
>  - 2페이즈 5분
>  - 3페이즈 3분
>  - 4페이즈 2분
>  - 5페이즈 1분
>  - 6페이즈(서든데스) 모두가 지상으로 이동되어 위더에 걸리고 최후의 승자를 가림
